# TLx Contracts
These are smart contracts for TLx projects (as needed). 

## Install dependencies
Run `make update`

## Build
Run `make build`

## Test
### Default Test
Run `make default_test`

### Gas Test
Run `make gas_test`

### Coverage Test
Run `make coverage_test`

### Fuzz Test
Run `make fuzz_test`

### Format code
Run `make fmt`
